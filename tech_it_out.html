Embedded system design event.

Tracks – 2 (Novice – FY, SY students, Expert – TY, BTech students) It is an embedded system design event in which the participants would be asked to design an embedded system on the given problem statement. There would be 3 rounds for this event. Each round would end with the elimination process. Software and hardware skills would be demanded for the qualification to the next rounds. A good hand at solving the problem statements, a strong hold over some technical knowledge and a necessary competitive spirit would definitely make the embedded system work at par!

No. of Rounds - 2
Fees - 80 + 150*

*Additional fees to be paid after the shortlisting

Team Size - 2 Participants per Team
Mode of event - offline

Round 1- TechWiz

  • This is a buzzer round. Also, to test the skills of Participants There will be analysis of basic problem statement.

Round 2 - Testbench

  • Round 2.1
    Participants will be provided with problem statement. They have to do the simulation and upload it on virtual hardware and implement it.

    Round 2.2
